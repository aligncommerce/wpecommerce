== Description ==

Add alligncommerce Payment Gateways for ecommerce
ecommercebacked shopping experience into a selling machine. This ecommerce gateway plugin, combined with a merchant account from alligncommerce Payment Gateways,
will allow you to accept transactions from alligncommerce Payment Gateways with bank tranfer and Bitcoin

== Installation ==

Installing The Payment Gateway Plugin

1.Download the plugin zip file.
2.Login to your WordPress Admin. Click on Plugins | Add New from the left hand menu.
3.Click on the “Upload” option, then click “Choose File” to select the zip file from your computer. Once selected, press “OK” and press the "Install Now" button.
4.Activate the plugin.
5.Open the settings page for store and click the "payments" tab.
6.Configure your settings.
